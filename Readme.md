# Nombre del proyecto

Barberia "Pepe"

# Objetivo General

Desarrollar un sistema web que permita gestionar las citas de la barbería 
de forma organizada y digital.

# Arquitectura

El sistema utiliza arquitectura cliente-servidor.
Django maneja la lógica del backend.
Firebase gestiona la base de datos.
HTML y CSS estructuran y diseñan la interfaz.

# Autores

Yeison David Moreno Nieto, Kevin Santiago Larrota, Juan David Cuadros.

# Tecnologias usadas

-Python
-Django
-Firebase
-HTML
-CSS

# Requisitos Funcionales del Cliente

Iniciar sesión.
Agendar cita.
Cancelar cita.
